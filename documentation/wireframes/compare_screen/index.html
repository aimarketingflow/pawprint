Pawprinting V2 - Compare Screen Wireframes

Overview

This documentation presents wireframes for a new dedicated "Compare" screen in the Pawprinting V2 application. Moving comparison functionality out of the Analyze screen into its own main navigation tab will improve the user experience by:

  • Providing a dedicated interface optimized specifically for comparison tasks
  • Simplifying the mental model of the application (Generate → Analyze → Compare)
  • Allowing more robust comparison features without overcrowding the Analyze screen
  • Creating space for specialized comparison visualizations and reporting tools

Implementation Plan

Design Decision

This wireframe set proposes moving comparison functionality into its own dedicated screen. This is a significant change from the current implementation where comparison is embedded in the Analyze screen.

1 Create CompareScreen Class

Develop a new CompareScreen class similar to other main screens (HomeScreen, GenerateScreen, AnalyzeScreen).

2 Implement File Selection Widget

Create a file selection sidebar component that allows users to select multiple pawprint files and organize them by origin or other criteria.

3 Develop Comparison Visualization

Port existing comparison functionality from AnalyzeScreen and enhance it with dedicated comparison visualizations and charts.

4 Add Tab Navigation

Implement tab-based navigation within the Compare screen for different view types (comparison, charts, raw data, summary).

5 Create Filtering and Export Options

Develop dialogs for filtering comparison results and exporting reports in various formats.

6 Update Main Navigation

Add Compare as a main navigation option in the application header alongside Generate and Analyze.

7 Remove Redundant Functionality

Once the dedicated Compare screen is implemented, remove the comparison button and related functionality from the Analyze screen.
