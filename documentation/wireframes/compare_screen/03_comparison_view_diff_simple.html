Features of the Simple Diff Display:

  • Line-based Diff: Traditional line-by-line comparison with color coding
  • Path Navigation: Clear JSON paths to locate the changes
  • Display Options: Toggle between different diff visualization types
  • Interactive Elements: Expand/collapse sections, copy paths
  • Color Coding: Green for additions, red for deletions, gray for unchanged content

The Simple Diff Display is one of several ways to visualize differences between pawprint files in the Compare screen.
