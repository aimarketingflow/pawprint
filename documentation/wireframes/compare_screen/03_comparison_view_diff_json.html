Features of the JSON Tree Diff:

  • Hierarchical View: JSON is displayed as an expandable/collapsible tree structure
  • Change Highlighting: Color coding shows added, removed, and changed elements
  • Change Values: Changed values show both the old and new values
  • Interactive Navigation: Expand/collapse sections to focus on areas of interest
  • Summary Stats: Counts of additions, deletions, and modifications

The JSON Tree Diff provides a more structured view of changes compared to the line-by-line diff, making it easier to navigate complex JSON structures.
