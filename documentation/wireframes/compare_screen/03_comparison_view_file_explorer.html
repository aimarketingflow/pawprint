Click on a file to see detailed changes

Features of the File Explorer:

  • Tabs: Quickly filter by change type (added, deleted, modified)
  • Search: Find specific files by name or path
  • Visual indicators: Color-coded badges and icons for each change type
  • Selection: Click any file to view its detailed changes (highlighted in blue)
  • Group By: Organize files by directory path for better navigation
  • Expand/Collapse: Control the view of nested directories

When a user clicks on a file in this explorer, the File Detail View (03_comparison_view_file_detail.html) will be displayed.
