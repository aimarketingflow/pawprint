Features of the Side-by-Side Diff View:

  • Parallel Comparison: Shows before and after content in parallel columns
  • Clear Highlighting: Color coding shows added, removed, and unchanged lines
  • Context Preservation: Maintains the context around changed lines
  • Navigation: Quick access to next/previous differences
  • Path Information: Shows the full JSON path for the current change
  • File Metadata: Displays file source and line numbers for reference

The Side-by-Side Diff View provides a traditional and familiar way to visualize differences between versions, making it easy to spot changes in their original context.
