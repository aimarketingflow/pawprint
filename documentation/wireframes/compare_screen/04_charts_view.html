Features of the Charts View:

  • Overview Cards: Quick visualization of key pattern scores with before/after indicators
  • Multiple Chart Types: Support for different visualization methods (radar, bar, line, etc.)
  • Contextual Information: Chart descriptions and insights to explain the significance of changes
  • Interactive Elements: Tabs for different pattern focuses and export options
  • Change Indicators: Visual markers highlighting significant differences between pawprints

The Charts View provides data-driven insights into the differences between pawprints, helping users identify and understand significant changes across patterns.
