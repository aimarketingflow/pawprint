Origin-Based Comparison Logic

Why Group by Origin?

Comparing pawprints from the same origin (source directory) offers more meaningful insights:

  • Files from the same directory structure have contextual relevance
  • Changes over time in the same directory are more significant
  • Pattern differences in the same structure reveal important insights
  • Reduces noise from comparing completely unrelated directories

Grouping Algorithm

# Pseudocode for origin grouping def group_pawprints_by_origin(pawprint_files): origin_groups = {} for file_path in pawprint_files: # Load JSON data data = load_json(file_path) # Extract origin from metadata if 'metadata' in data and 'origin' in data['metadata']: origin = data['metadata']['origin'] # Add to appropriate group if origin not in origin_groups: origin_groups[origin] = [] origin_groups[origin].append({ 'file_path': file_path, 'data': data }) # Only return groups with multiple files return {k: v for k, v in origin_groups.items() if len(v) > 1}

UI Components for Comparison

Overview Tab

The comparison view features multiple UI components organized in a tab interface:

  • Overview Tab: Shows origin grouping and file listings
  • Origin Group Buttons: One button per common origin with multiple files
  • All Files List: Complete listing of all selected files
[Overview Tab Screenshot - Shows origin grouping with buttons and file listing]

Detailed Comparison Dialog

When an origin group is selected, a detailed comparison dialog appears:

  • Differences Tab: Shows all differences between files with highlighting
  • File-Specific Tabs: One tab per file showing complete JSON content
  • Export Button: Allows saving the comparison as HTML
[Comparison Dialog Screenshot - Shows differences tab with highlighted variances]

HTML Export Format

The exported HTML report includes:

  • AIMF LLC branding and logo
  • Summary information about compared files
  • Color-coded differences with file origins
  • Complete JSON data for each file in collapsible sections
  • Timestamp and origin metadata
